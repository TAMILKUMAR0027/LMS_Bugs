## Preconditions
- At least one deletable course category exists.

## Steps to Reproduce
1. Navigate to Dynamic Field Settings > Course Category tab.
2. Delete an existing course category.
3. Refresh the page.
4. Verify whether the deleted category is still listed.
---
## Expected Result
Once deletion is confirmed, the category should be permanently removed and must not reappear after a page refresh; the delete operation should complete within an acceptable response time.
 
---
## Actual Result
The deleted category still shows up in the list after refreshing the page; deletion appears to take significantly longer than expected to reflect, or does not persist as removed.

## Impact
This indicates the delete operation is either delayed on the backend or the UI is not correctly re-syncing with the latest data after deletion — a data-integrity concern that could allow deleted records to be re-used or displayed to end users incorrectly.
