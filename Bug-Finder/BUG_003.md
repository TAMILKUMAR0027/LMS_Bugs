## Steps to Reproduce
1. Navigate to Course Structure > Add Course > Pedagogy tab.
2. Open the 'I Do' (or similar) dropdown and select an option.
3. Close the dropdown (click outside / select item).
4. Attempt to interact with the rest of the page/background.
---
## Expected Result
After selection/closure, the dropdown overlay should fully close and background elements should be interactable immediately, with proper z-index stacking.
 
---
## Actual Result
The dropdown overlay remains visually/functionally on top of the background after it should have closed, blocking further interaction with the underlying form.

## Impact
When adding a course and interacting with a dropdown inside the Pedagogy section (e.g., 'I Do' multi-select list), the overlay's z-index remains above page content after it should have closed, preventing subsequent actions and blocking the Add Course workflow.
