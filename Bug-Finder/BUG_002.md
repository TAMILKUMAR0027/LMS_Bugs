## Preconditions
- A course record is present/searched in the Course Structure grid.
---
## Steps to Reproduce
1. Navigate to Course Structure page.
2. Search/locate a course record.
3. Click the action (three-dot) icon on the row via automation.
4. Attempt to select 'Edit Course' from the dropdown.
---
## Expected Result
The action dropdown should render in the DOM with correct visibility/attributes so it can be located and interacted with reliably, both manually and via automation.
 
---
## Actual Result
Dropdown does not render as an interactable/visible element for automation (element not found / not clickable), causing the test step to fail, although manual clicking shows no visible issue.

## Impact
Clicking the action (three-dot) button on a searched course record should open a dropdown with 'View Full Details' and 'Edit Course' options. The dropdown does not become visible/interactable in the DOM during automated execution, blocking automation coverage of the Edit/View flows even though the feature itself works manually.
