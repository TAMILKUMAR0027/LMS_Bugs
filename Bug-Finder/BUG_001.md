## Steps to Reproduce
1. Navigate to Course Structure page.
2. Enter a valid search keyword (e.g., course name/ID) in the search bar.
3. Trigger search via automation (repeat run 5–10 times).
4. Observe filtered grid results each run.
---
## Expected Result
Search should consistently filter and display matching records every time the same query is executed, with a stable, predictable response time.
 
---
## Actual Result
Search returns correct filtered results intermittently; on other executions it returns unfiltered/stale/empty results for the identical query.

- [x] Intermittent

## Impact
The keyword search field on the Course Structure listing page does not consistently filter/return results, indicating a non-deterministic search/filter behavior (likely a race condition between debounce/API response and grid re-render). This undermines confidence in search results and destabilizes automation suites.
